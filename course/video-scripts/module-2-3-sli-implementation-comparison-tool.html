Implementation Trade-offs

Metric-based SLIs provide faster calculation but may lose granular detail compared to log-based analysis.

Metric-Based Pros
  • Real-time calculation
  • Low storage cost
  • Fast alerting
  • Easy aggregation
Log-Based Pros
  • Full context available
  • Complex filtering
  • Historical analysis
  • Rich metadata

Log Parsing Rules

Extract: status, duration, method, path
Filter: status < 500 (success criteria)
Aggregate: success_count / total_count * 100

Log-Based Approach

Detailed log analysis provides rich context and flexible SLI calculation from raw event data.

Best for: Complex filtering, historical analysis, detailed debugging, compliance requirements.
