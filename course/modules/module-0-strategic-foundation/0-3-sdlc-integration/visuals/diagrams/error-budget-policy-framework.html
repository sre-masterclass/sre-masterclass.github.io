Error Budget Policy Framework

Policy Components

1. SLO Definition & Measurement

Service Level Objectives:
- Checkout Success Rate: 99.95%
- API Response Time: 95th %-ile < 200ms
- System Availability: 99.9% uptime

2. Error Budget Calculation

Mathematical Definition:
(1 - SLO) × time period

Examples:
- 99.95% SLO = 22 mins/month budget
- 99.9% SLO = 43 mins/month budget

3. Error Budget Consumption Triggers

  • Green Status (<25% consumed)
    Action: Normal development velocity.
    Deployment: Automated approvals enabled.
    Focus: Feature development.
  • Yellow Status (25-75% consumed)
    Action: Increased reliability focus.
    Deployment: Manual reliability review required.
    Focus: Balance features and reliability work.
  • Red Status (>75% consumed)
    Action: Development freeze activated.
    Deployment: Only critical fixes allowed.
    Focus: Restore service reliability.
