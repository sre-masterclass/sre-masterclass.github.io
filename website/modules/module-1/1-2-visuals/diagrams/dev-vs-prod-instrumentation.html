Development Instrumentation

Focus: Maximum detail, debugging ease. Assumes stable resources.

Deep Instrumentation Example:

# Comprehensive business logic metrics
http_requests_total = Counter(
    'http_requests_total',
    'Total HTTP requests',
    ['method', 'status', 'endpoint', 'user_type', 'region', 'version']
)

business_operations_total = Counter(
    'business_operations_total',
    'Business operations by type',
    ['operation', 'result', 'customer_tier', 'payment_method', 'product_category']
)

Production Instrumentation

Focus: Reliability, performance, and cost-efficiency. Assumes resource contention.

Shallow Instrumentation Example:

# Essential HTTP metrics only
http_requests_total = Counter(
    'http_requests_total',
    'Total HTTP requests',
    ['method', 'status']
)

http_request_duration = Histogram(
    'http_request_duration_seconds',
    'HTTP request duration',
    ['method']
)
